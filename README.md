# 🎓 Warana University Management System (UMS)

A modern, scalable **University Management System** designed to handle admissions, colleges, departments, and student data with a centralized Super Admin control panel.

---

## 🚀 Project Overview

The Warana University Management System is a multi-level web application that manages the complete university workflow:

* Admission processing
* College and department management
* Student data tracking
* Eligibility verification
* Centralized admin control

---

## 🏗️ System Architecture

The system follows a hierarchical structure:

```
Super Admin (University Level)
        ↓
College Admin
        ↓
Department Module
        ↓
Eligibility Cell
        ↓
Student Portal
```

---

## 👤 User Roles

| Role             | Description                      |
| ---------------- | -------------------------------- |
| Super Admin      | Manages entire university system |
| College Admin    | Manages college-level operations |
| Department       | Handles department processes     |
| Eligibility Cell | Verifies student eligibility     |
| Student          | Accesses academic services       |

---

## 💎 Features

### 🔹 Super Admin Module

* Manage colleges (Add/Edit/Delete)
* View departments and students
* Dashboard analytics (charts, stats)
* Central system control

### 🔹 College Module

* Manage departments
* Track student data
* Course management

### 🔹 Department Module

* Process student applications
* Manage academic activities

### 🔹 Eligibility Cell

* Verify documents
* Approve/Reject applications

### 🔹 Student Module

* Login portal
* Profile management
* Application tracking
